Day 5

IT#28 StarterPackCamp

วันนี้เป็นวันสุดท้ายแล้วที่เราจะได้เข้าค่ายแบบออนไซต์เพราะว่ามีนักศึกษาติดโควิด-19 เป็นเรื่องที่น่าเสียใจมากเพราะผมอยากมาเข้าค่ายที่มหาลัยสุดๆ แต่เนื่องจากการเกิดเหตุการณ์ดังกล่าวทำให้พี่ที่จัดค่ายได้ปรับรูปแบบ การเรียน การสอน เป็นออนไลน์ทั้งหมดแทน โดยกลุ่มที่ผมชื่อมีชื่อว่า เฮอริเคน ซึ่งเป็นชื่อของเครื่องเล่นในสวนสนุก พอได้มาเข้าค่ายแบบนี้แล้วผมรู้สึกสนุกมากเลย อีกทั้งยังได้ความรู้จากพี่ๆ ที่สอนอีกโดยวันนี้เราก็มีการเรียนตามปกติละก็มิพักเบรคและกิจกรรมเล่นเกมกันเล็กน้อย พอจบการเรียน การสอน ของวันนี้พวกพี่ที่ดูแลพวกเราก็เรียกถ่ายรูปรวมทุกคนที่มาร่วมกิจกรรมและถ่ายแยกเป็นกลุ่มเครื่องเล่นต่างๆ ด้วยจากนั้นก็แยกย้ายกันกลับบ้าน

แค่คิดว่าจะต้องกลับบ้านก็ทำให้ผมนึกถึงวันแรกที่มาค่ายเลย ตอนมาผมก็งงๆ ละก็ไม่มีเพื่อนเลยซักคนครับแต่พอได้มาเข้าค่ายทำให้ผมได้รู้จักเพื่อนๆ พี่ๆ อีกเยอะแยะเลย โดยในวันแรกพี่ๆ เขาก็ให้เราคิดท่ากลุ่มของตัวเอง กลุ่มของผมก็ได้ท่าของ Dr.Strange เป็นท่าหมุนๆ เปิดประตูมิติ ในระหว่างเรียนผมก็ได้รู้จักกับเพื่อนที่นั่งข้างผม เขาชื่อว่าเล้ง ตอนคุยกันเล้งดูฉลาดและเก่งมากเลยครับแต่พอผมหันไปเรียนแล้วซักพักผมหันมาอีกที เล้งก็หลับไปแล้วครับ การได้เข้าค่ายนี้ทำให้ผมประทับใจมากเลย จนทำให้ผมชอบในการเข้าค่ายมากเลย
