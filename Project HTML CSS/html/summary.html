แท็ก video

เป็นแท็กที่ใช้ในการใส่วิดีโอ ส่วน control คือปุ่มไว้ควบคุมการเล่นวิดีโอ เช่น ปุ่ม play, pause ฯลฯ ส่วน source เป็นแท็กใช้เพื่อระบุแหล่งข้อมูลสื่อต่างๆ เช่น วิดีโอ เสียง และ ภาพ ส่วน type คือให้บอกว่าวิดีโอนั้นมีนามสกุลไฟล์อะไร เช่น mp4, webm ฯลฯ ข้อความ "Your browser does not support the video tag." จะแสดงผลในกรณีที่บราวเซอร์ของผู้ใช้นั้นไม่ support วิดีโอเพราะฉะนั้น ถ้าบราวเซอร์ไหนไม่ support มันก็จะไม่เห็นเป็นไฟล์วิดีโอ แต่ขึ้นข้อความบอกให้ผู้ใช้รู้แทนว่าบราวเซอร์ของคุณไม่ support ถ้าอยากจะให้มัน support ทั้งหมดก็ใส่หลายๆ นามสกุลเลย บราวเซอร์ไหน support นามสกุลไหนก็จะทำการดึงขึ้นมาแสดงเอง
