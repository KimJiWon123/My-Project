Day 4

Benjakitti Park

วันนี้ผมได้ไปสวนเบญจกิตินี่เป็นครั้งแรกเลยที่ได้ไปเดินเล่นในสวนกับคนพิเศษ เราได้มีการตกลงไว้ว่าจะไปช่วงบ่ายๆเย็นๆเพราะคิดว่าอากาศกำลังดีไม่ร้อนมาก เราสองคนเจอกันที่ป้ายรถเมล์และต่อ BTS ไปลงสถานีอโศก และเดินไปอีกซักพักนึงก็ถึงสวนแล้ว เราสองคนได้แวะซื้อของกินที่ประตูทางเข้าสวนแล้วมานั่งกินกัน จากนั้นก็ไปถ่ายรูปเล่น พอชมบรรยากาศรอบๆสวนเสร็จเราก็เดินไปที่ห้าง Terminal 21 Asok หาของกินก่อนกลับบ้านกัน ไม่แปลกใจที่คนมาสวนแห่งนี้กันเยอะ เพราะมีดอกไม้หลากหลายสี มีสระน้ำแอ่งใหญ่ มีที่ให้นั่งเล่นนั่งพัก มีสะพานยาวมากๆ ไว้ให้คนได้ถ่ายรูป ได้วิ่งออกกำลังกาย วันนี้ก็เป็นอีกวันที่ผมได้ใช้เวลาร่วมกับคนพิเศษและมีความสุขมากๆ เลยครับ
